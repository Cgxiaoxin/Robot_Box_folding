## 1 概述

本文档详细描述了镜头电机控制系统的CAN通讯协议。该系统基于STM 32微控制器,使用CAN总线进
行通讯,支持多种控制模式和参数配置。
本文档所有数据相关单位:位置信息:弧度,速度信息:弧度/秒,电流信息:安培,电压信息:伏
特,温度信息:摄氏度,扭矩信息:牛米
## 2 协议格式

### 2.1 基本帧格式

标准CAN帧格式:
-  ID(11位):设备地址标识
-  DLC:数据⻓度码
-  广播包:Data[0-64];标准包:Data[0-7];数据字段
### 2.2 命令帧格式

## 1 广播包命令帧:

命令帧由以下部分组成:
-  Data[0]:命令码(CM D)
-  Data[1-7]:参数数据
## 2 标准包命令帧

-  Data[0-64]:参数数据
📢仅M IT控制支持广播包,故广播包在数据段内无CM D字段,只要是广播包ID,则默认按照MIT
控制协议解析命令

int util_float2Uint(float x, float x_min, float x_max, int bits){
/// Converts a float to an unsigned int, given range and number of bits ///
float span = x_max - x_min;
float offset = x_min;
return (int) ((x-offset)*((float)((1<<bits)-1))/span);
}
float util_uint2Float(int x_int, float x_min, float x_max, int bits){
/// converts unsigned int to float, given range and number of bits ///
float span = x_max - x_min;
float offset = x_min;
return ((float)x_int)*span/((float)((1<<bits)-1)) + offset;
}
